# 💬 Fake WhatsApp Chat Application

A Full Stack WhatsApp-inspired Chat Application built with **Node.js**, **Express.js**, **MongoDB**, and **EJS**. The application allows users to create, edit, delete, and view chat messages through a simple and responsive interface.

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/sarita2233/Fake-Whatshapp.git
```

### 2. Move into project folder

```bash
cd Fake-Whatshapp
```

### 3. Install dependencies

```bash
npm install
```

### 4. Create a .env file

```
MONGO_URI=your_mongodb_connection_string
```

### 5. Run the application

```bash
node index.js
```

or

```bash
nodemon index.js
```

Open

```
http://localhost:8080/chats
```
